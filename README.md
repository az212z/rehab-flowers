## سبب الاختيار
نشاط ورد وهدايا بتقييم عالٍ وموثّق (4.9 بأكثر من 573 مراجعة) في بريدة، وبلا موقع إلكتروني — فرصة قوية لإطلالة رقمية فاخرة تعكس جودة المنتج وتُسهّل الطلب والتواصل.

## فكرة التصميم
هوية **"وردي راقٍ فاخر"** تميّزها عن باقي مواقع الورد:
- **الألوان:** وردي مغبّر (#C98B9E) + خمري عميق (#5E2433) + كريمي دافئ (#F7EFE9) + لمسة ذهبية (#C7A05A).
- **الخطوط:** عناوين "El Messiri" + نص "Tajawal".
- إحساس راقٍ ومرتفع المستوى، مع حركات دقيقة (ظهور تدريجي، تكبير الصور، تقلّص الهيدر).
- **نسخة محايدة الجنس** بالكامل في كل النصوص (اطلب باقتك، تواصل معنا...).
- **قائمة الجوال = طبقة كاملة معتمة (full-screen overlay)** بزر إغلاق X واضح — وليست درج جانبي.

## التقنيات
HTML / CSS / JavaScript خام (vanilla) — بدون أي إطار عمل أو مكتبات، ولا خطوة بناء. يعمل مباشرة على GitHub Pages.

## الصور
الصور حقيقية للمنتجات من ملف المشروع (مصدرها خرائط قوقل للنشاط). **تم استبعاد** أي صور تحمل لافتة لنشاطٍ آخر (Razan / Eli Roes / غيرها) والاكتفاء بصور المنتجات النظيفة الخاصة بالورد:
- `rh-11971.jpg` — باقة ورد طبيعي مشكّلة.
- `rh-14736.jpg` — تنسيق ورد أصفر فاخر في إناء خزفي.

## حالة الـ backend
**غير مربوط بخادم.** نموذج الطلب يجهّز رسالة واتساب مسبقة التعبئة عبر `wa.me` ويحفظ نسخة محلية في `localStorage` (تجريبي)، مع إشعار نجاح (toast).
